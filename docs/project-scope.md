# Project Scope

## Ziel

Dieses Repository soll eine reproduzierbare, schrittweise nachvollziehbare Installation von PVE / Proxmox-ähnlichen ARM64-Komponenten auf Raspberry Pi 4 und Raspberry Pi 5 ermöglichen.

## Kein offizielles Proxmox

Dieses Projekt ist kein offizielles Proxmox VE Projekt und wird nicht von Proxmox unterstützt.

## Fokus

- Raspberry Pi 4 ARM64
- Raspberry Pi 5 ARM64
- Debian/Raspberry Pi OS 64-bit Basis
- Nachvollziehbare Scripts
- Saubere Checks vor und nach Änderungen
- Dokumentierte Einschränkungen

## Nicht-Ziele

- Kein produktiver Ersatz für offizielles Proxmox VE auf x86
- Kein garantierter Upgrade-Pfad
- Kein Support für alle PVE Enterprise-Funktionen
